MIDI to Cirklon Converter

Convert pencilresearch/midi CSV files to Sequentix Cirklon .cki instrument definitions

How it works: This tool converts MIDI CC definitions from the pencilresearch/midi repository into Cirklon .cki instrument definition files for use with the Sequentix Cirklon sequencer.

Local Database: Includes 279 instruments from 88 brands cached locally for instant access. Use "Update Instrument Data" to refresh with the latest data from the repository.

Character Limits: The tool automatically handles Cirklon's strict limits: instrument names (9 chars max), CC parameter labels (6 chars max). Long names are shortened by removing spaces first, then lowercase vowels, while always preserving capital letters.

Name Review: Check "Review and customize parameter names" to see all shortenings and provide your own custom names before generating the .cki file.

Cirklon Instrument Settings: Configure MIDI port, channel, default note (C3), pattern type (P3), and other advanced options compatible with the Cirklon Organizer tool. Multi, Presend Program, and Poly Spread are boolean on/off settings.
